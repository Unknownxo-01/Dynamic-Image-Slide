🖼️ Dynamic Image Slider

Dynamic Image Slider is a web-based application that allows users to view and navigate through a collection of images interactively. The slider automatically transitions between images and also provides manual navigation controls (like next/previous buttons or indicators).

This project demonstrates the use of React.js, HTML, CSS, and JavaScript to build an engaging and responsive image carousel. It dynamically loads images and ensures smooth transitions, making it suitable for photo galleries, portfolios, or product showcases.

✨ Key Features

Automatically sliding images with smooth transitions

Manual navigation using arrows or buttons

Responsive design that adapts to all screen sizes

Easy to customize image sources and transition speed

Lightweight and efficient front-end implementation

💻 Technologies Used

React.js – for dynamic component rendering

HTML5 – for structure

CSS3 – for styling and animations

JavaScript (ES6) – for interactivity
